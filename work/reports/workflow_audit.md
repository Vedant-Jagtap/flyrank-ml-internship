# AI Workflow Audit

| Task | Classification | Rationale |
|------|----------------|-----------|
| Study Machine Learning concepts | Collaborate with AI | AI explains concepts, but I verify and practice them myself. |
| Write Python code | Collaborate with AI | AI helps generate code, but I test and debug it. |
| Debug Python errors | Delegate to AI with review | AI suggests fixes; I verify before applying them. |
| Read research papers | Delegate to AI with review | AI summarizes papers, but I read important sections myself. |
| Prepare internship reports | Delegate to AI with review | AI improves structure and grammar while I verify accuracy. |
| Resume improvement | Delegate to AI with review | AI suggests improvements, but I choose the final content. |
| GitHub documentation | Delegate to AI with review | AI drafts documentation; I check technical correctness. |
| Brainstorm ML project ideas | Collaborate with AI | AI provides ideas while I evaluate feasibility. |
| Interview preparation | Collaborate with AI | AI simulates interviews and explains concepts. |
| Professional email writing | Fully automate | AI can draft professional emails with minimal edits. |
| Weekly study planning | Fully automate | AI creates schedules that I review and follow. |
| Official internship submissions | Just me | I am responsible for all official submissions. |
| Final project decisions | Just me | Project direction and technical decisions are my responsibility. |

---

# Three Target Tasks

## 1. Debug Python Code

Success:
- Root cause identified.
- Error resolved.
- No new bugs introduced.

## 2. Summarize Research Papers

Success:
- One-page summary.
- Covers objective, methodology, results, limitations, and future work.

## 3. Machine Learning Model Development

Success:
- Model is reproducible.
- Evaluation metrics documented.
- Code is clean and understandable.
